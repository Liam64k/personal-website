# Liam Stier — Personal Website

A clean, responsive personal website built with HTML, CSS, and JavaScript.

## Project Structure

```
personal-website/
├── assets/             # Media and static assets
│   ├── fonts/          # Custom web fonts
│   ├── icons/          # Favicon and icon files
│   └── images/         # Image files
├── public/             # Public root files (robots.txt, sitemap, etc.)
├── src/                # Source code
│   ├── css/            # Stylesheets
│   │   └── styles.css
│   ├── js/             # JavaScript
│   │   └── main.js
│   └── index.html      # Main HTML page
├── dist/               # Production build output (git-ignored)
├── build.js            # Build script
├── package.json        # Project metadata and scripts
└── README.md
```

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or later)

### Installation

```bash
git clone https://github.com/Liam64k/personal-website.git
cd personal-website
npm install
```

### Development

To start a local development server with live reload:

```bash
npm run dev
```

### Production Build

To create an optimized production build in the `dist/` folder:

```bash
npm run build
```

## Tech Stack

- **HTML5** — Semantic markup with SEO best practices
- **CSS3** — Custom styles, responsive design
- **JavaScript** — Vanilla JS, no frameworks
